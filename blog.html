What is the Document Object Model (DOM)?

Dom Means Document Object Model.It Works As a tree of Object.By DOM We can interact and Manipulate the HTML and CSS of a Webpage.

How do you select an element from the DOM?

To Select a element from a DOM we need to set a id or class in the item what we need to target and select it by getElementId/Class and Have to add addEventListener for funtionality

What is event delegation in the context of the DOM, and why is it useful?

By Event delegation we can select a single event listener and get a specific child from parent.Its Useful Because we can attach a single event listener to a parent. This reduces memory usage and improves performance.
